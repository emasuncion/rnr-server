## Developer Notes
* JavaScript coverage report is located in `tests/coverage`.
* When viewing swagger-ui on `http://localhost:3333/docs`, it is expected to have an "Invalid" error message at the bottom right corner. This is a [known issue](https://github.com/ahmadarif/adonis-swagger/issues/5). To fix this, put `http://localhost:3333/swagger.json` in the `Explore` field.
